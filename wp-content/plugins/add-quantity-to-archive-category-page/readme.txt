A small plugin which lets you add a quantity field to woocommerce product category/archive page.

== Description ==
It does what it says. Simple plugin to add a quantity field to woocommerce category/archive page. Your customers can add the quantity straight from the archive/category page.

What do you need to do to make it work?
1. Download the plugin (If you are not installing directly from Wordpress)
2. Upload the plugin on to Wordpress 
3. Activate the plugin

That's pretty much it! Your plugin is ready and you should see a quantity field on woocommerce category/archive page.

== Installation ==
1. Upload \"test-plugin.php\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.


== Frequently Asked Questions ==
= Why are the quantity and add to cart buttons on different lines? =
Depending on your theme it can be on different locations but should be a quick css fix. If you are struggling just send me a message.

=How do I take the field off now?=
Simple disable the plugin.
